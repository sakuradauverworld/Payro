# 一気に送る君

給与明細などのPDFを、宛先ごとに自動でメール送信するWindowsアプリです。

## ダウンロード

| OS | ダウンロード |
|---|---|
| Windows | [IkkiniOkuru-windows.exe](https://github.com/sakuradauverworld/Payro/releases/latest/download/IkkiniOkuru-windows.exe) |
| macOS | [IkkiniOkuru-mac.zip](https://github.com/sakuradauverworld/Payro/releases/latest/download/IkkiniOkuru-mac.zip) |

## セットアップ

### 1. Gmailアプリパスワードの発行

送信に使うGmailアカウントで以下の手順を行ってください。

1. [Googleアカウント設定](https://myaccount.google.com/) を開く
2. 「セキュリティ」→「2段階認証プロセス」を有効にする
3. 「セキュリティ」→「アプリパスワード」を開く
4. アプリ名に「一気に送る君」などと入力して「作成」をクリック
5. 表示された16文字のパスワードをメモしておく

### 2. アプリの設定

1. ダウンロードした `IkkiniOkuru-windows.exe` をダブルクリックで起動する
   - 「WindowsによってPCが保護されました」と表示された場合は「詳細情報」→「実行」をクリック
2. 「ユーザー設定」タブでGmailアドレスとアプリパスワードを入力して「保存」をクリック

## 使い方

1. **グループ管理タブ** で送信先グループを作成し、宛先（氏名・メールアドレス）を登録する
2. **テンプレート管理タブ** でメール件名・本文を登録する
3. **送信タブ** でグループ・年月を選択し、PDFファイルをドラッグ＆ドロップして送信する

詳しい操作方法はアプリ内の「使い方」タブを参照してください。
