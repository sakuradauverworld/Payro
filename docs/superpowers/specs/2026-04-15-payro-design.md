# Payro 設計ドキュメント

**作成日:** 2026-04-15

## 概要

毎月従業員全員にPDF給与明細をメールで送信する作業を自動化するGUIデスクトップアプリ。
経理担当者が使いやすいよう直感的なUIを提供し、他の会社への配布を想定してEXEとして配布する。

---

## 要件

### 機能要件

- 複数のPDFファイルをドラッグ&ドロップまたはフォルダ選択で一括読み込みできる
- PDFファイル名から従業員を自動マッチングできる（パターンは設定でカスタマイズ可能）
- 従業員リスト（名前・メールアドレス・ファイル名パターン）をアプリ内で管理できる
- メール件名・本文をテンプレートで管理し、従業員名・月などを差し込める
- GmailのSMTPでメール送信する
- PDFなし従業員がいる場合は警告ダイアログを表示し、スキップして残りに送信できる
- 送信完了後に結果レポート（成功/失敗/スキップ）を表示・保存する

### 非機能要件

- EXEファイルとして配布できる（PyInstallerでビルド）
- Python・技術知識がない経理担当者でも使える
- マネーフォワード給与など複数のサービスのPDF命名規則に対応できる（設定で変更可能）

| 項目 | 技術 |
|------|------|
| 言語 | Python 3.11+ |

```
└── build/                   # PyInstallerのビルド出力
```

## 画面設計

### タブ構成

アプリはタブ形式で3つの画面を持つ。

#### 1. 送信タブ（メイン画面）

操作フロー：

1. PDFファイルをドラッグ&ドロップ、またはフォルダ選択ボタンでフォルダ内のPDFを一括読み込み
2. ファイル名から従業員を自動マッチングし、一覧表示
   - 緑行: PDFあり → 送信予定
   - 赤行: PDFなし → スキップ予定
3. 「送信」ボタンを押す
4. PDFなしの従業員がいる場合、確認ダイアログを表示
   - 例:「以下の2名はPDFがありません。スキップして残りの8名に送信しますか？」
5. OKで一括送信実行
6. 送信完了後、結果レポートを画面に表示（保存も可能）

#### 2. 従業員管理タブ

- 従業員一覧をテーブル表示（名前・メールアドレス・ファイル名マッチパターン）
- 追加・編集・削除ボタン
- CSVエクスポート・インポートボタン

#### 3. 設定タブ

- Gmailアカウント（メールアドレス・アプリパスワード）
- PDFファイル名マッチングパターン（正規表現またはキーワード方式）
- メール件名テンプレート（例：`{year}年{month}月分 給与明細`）
- メール本文テンプレート（例：`{name}様、{month}月分の給与明細を添付します。`）

## データ設計

### employees.csv

```csv
name,email,filename_pattern
山田太郎,yamada@example.com,山田太郎
佐藤花子,sato@example.com,佐藤花子
```

`filename_pattern` はPDFファイル名に含まれるキーワード。マネーフォワード等のファイル名形式に合わせて変更可能。

### config.json

```json
{
  "gmail_address": "company@gmail.com",
  "gmail_app_password": "xxxx xxxx xxxx xxxx",
  "subject_template": "{year}年{month}月分 給与明細",
  "body_template": "{name}様\n\n{month}月分の給与明細を添付いたします。\n\nご確認よろしくお願いいたします。",
  "filename_pattern_type": "contains"
}
```

### 送信レポート（テキストファイル）

```
送信日時: 2026-04-25 10:30:00
対象月: 2026年4月

成功 (8名):
  - 山田太郎 <yamada@example.com>
  - ...

失敗 (1名):
  - 鈴木一郎 <suzuki@example.com> - エラー: 接続タイムアウト

スキップ (1名):
  - 田中次郎 - PDFなし
```

## エラーハンドリング

| ケース | 対応 |
|--------|------|
| Gmail認証失敗 | エラーダイアログ表示（アプリパスワード設定を案内） |
| PDFファイルが読み込めない | 該当ファイルをスキップしてログに記録 |
| メール送信失敗（1件） | 失敗としてレポートに記録し、残りの送信を継続 |
| 従業員リストが空 | 送信前にチェックしてダイアログで通知 |

---

## 配布

- PyInstallerで単一EXEファイルにビルド
- EXEと同じフォルダに `data/` フォルダが作成され、設定・従業員リストが保存される
- 初回起動時はセットアップウィザードを表示（Gmailアカウント設定を案内）
